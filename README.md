
#     <center>    springboot博客    </center>


## 介绍:

###    基于springboot的博客项目

##     软件架构
####    1.	此博客基于springboot，前端页面通过thymleaf进行渲染。
####    2.	博客前端可以展示博客内容，分类标签，作者信息以及评论等功能。
####    3.	在博客编辑页面加入了markdown插件，可在线编辑markdown内容。
####    3.	博客后台可以实现登录功能，对博客的增删改查功能，以及管理评论，管理标签和分类等等。
####    4.	配置了日志监控，在生产环境下可以输出相关的日志。
